Windmark is a premium, high-performance job portal built with the latest web technologies. It features a stunning, glassmorphic interface, advanced real-time filtering, and a seamless user experience.

## ‚ú® Features

- **Premium UI/UX**: Modern glassmorphic design with subtle micro-animations and smooth transitions.
- **Advanced Filtering**: Robust, real-time filters for Job Category, Location, Employment Type, Remote, and Salary.
- **Infinite Scrolling**: Optimized infinite scroll with predictive early loading for a lag-free experience.
- **Theme Switching**: Seamless transition between sophisticated Light and Dark modes.
- **Data Export**: Export your filtered job lists instantly to **CSV**, **Excel**, or **PDF** formats.
- **Fast Search**: Instant, debounced search across job titles, companies, and descriptions.
- **Responsive Design**: A first-class experience across mobile, tablet, and desktop devices.

- **Core**: [Next.js 16](https://nextjs.org/) (App Router), [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)

- **Icons**: [Lucide React](https://lucide.dev/)
- **Utilities**: [TanStack Table](https://tanstack.com/table/latest), [PapaParse](https://www.papaparse.com/) (CSV), [jsPDF](https://github.com/parallax/jsPDF)

## üõ†Ô∏è Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed and [pnpm](https://pnpm.io/) for dependency management.

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/windmark.git
   cd windmark
   ```

2. **Install dependencies**

   ```bash
   pnpm install
   ```

3. **Run the development server**

   ```bash
   pnpm dev
   ```

4. **Open the app**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application in action.
